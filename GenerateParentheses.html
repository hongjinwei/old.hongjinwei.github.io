Generate Parentheses

@https://leetcode.com/problems/generate-parentheses/

Given n pairs of parentheses, write a function to generate all combinations of well-formed parentheses.

For example, given n = 3, a solution set is:

"((()))", "(()())", "(())()", "()(())", "()()()"

实际上还是递归产生全排列的问题，只是在选取一个括号放入已产生的队列中的时候，要判断是否能放入这个括号。

要求放入的括号能够成功匹配，即“)”放入时，要检查是否有一个“)”与之对应。

我们用一个count变量来表示对应关系。每放入一个“（”，count++,表示有一个可以对应的“（”，放入一个“）”时count--，表示能对应的“（”减少一个。那么自然，当count=0时，放入“）”肯定是不符合规则的。

另外，再用lnumber，rnumber分别来表示未放入的左右括号数量，那么核心的判断函数puttable()的逻辑还应该包括是否还有括号可以取用。（我们用index=1表示左括号，index=2表示右括号）。

还有，我们用remain变量来统计剩下的括号数量，当remain==0的时候，表示已经存完，保存序列。

最后需要注意的是，在调用递归之后，必须要将新添加的括号从已产生队列中删除（由于我们这里才用的是随机选取括号放入队列，因此每次只需把最后一个括号删除即可，记得count 和lnumber/rnumber要跟着变动）。

我的AC代码：

public class Solution {
    private List<String> result = new LinkedList<String>();
    int count=0;
    int lnumber;
    int rnumber;
    private LinkedList<String> tmp = new LinkedList();

    private boolean puttable(int index)
    {
        if(index == 2){//')'
            if(count <= 0){//no '(' to
                return false;
            }else if(rnumber <= 0){
                return false;
            }else{
                return true;
            }
        }

        if(index == 1){//'('
            if(lnumber <= 0){//no '(' left
                return false;
            }else{
                return true;
            }
        }
        return true;
    }

    private void put(int index){
        if(index == 1){
            tmp.add("(");
            count++;
            lnumber--;
        }else{
            tmp.add(")");
            count--;
            rnumber--;
        }
    }

    private void remove(int index){
        tmp.pollLast();
        if(index ==1){
            lnumber++;
            count--;
        }else{
            rnumber++;
            count++;
        }
    }

    public void formatString(int remain)
    {
        if(remain == 0){
            String s = "";
            for(String x:tmp){
                s += x;
            }
            result.add(s);
        }else{
            for(int i=1; i<=2;i++){
                if(puttable(i)){
                    put(i);
                    formatString(remain-1);
                    remove(i);
                }
            }
        }
    }

    public List<String> generateParenthesis(int n) {
        rnumber = lnumber = n;
        formatString(2*n);
        return this.result;
    }

}
